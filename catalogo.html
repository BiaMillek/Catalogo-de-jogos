Projeto didatico com Flask + Fetch

Catálogo de Jogos

Exemplo de front-end com HTML, CSS e JavaScript consumindo dados JSON de um back-end Flask.

Destaque Catálogo Contato
HTML + CSS + JS +Flask

Uma base simples para ensinar cliente, servidor e consumo de API

O front-end buscara os jogos com fetch('/api/games') e enviara o formulário para /api/contact.

O que este exemplo mostra

  • Estrutura semantica com HTML
  • Layout com Grid e Flexbox
  • Manioulação de DOM e eventos
  • Consumo e envio de JSON com Fetch API
